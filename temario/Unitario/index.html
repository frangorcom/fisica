Vector unitario

Un vector unitario, también conocido como vector normalizado, es un vector que tiene dirección y sentido. No tiene dimensión y su magnitud o módulo es igual a uno este comuúnmente representado como: \(\vec{|\hat{v}|}=1\)

También el vector unitario puede representarse como: \(\hat{v}=\frac{\vec{v}}{\vec{|v|}}\)

Este vector nos permite representar la dirección y sentido de cualquier vector de forma prácica, facilitando cálculos como el del ángulo resultante entre 2 vectores, cabe señalar que el vector unitario al igual que cualquier vector puede pertencer a cualquier dimensión de \(N\leq 2\)
