Vector de posición

En Física, la posición, vector de posición o vector posición de un cuerpo respecto a un sistema de referencia se define como el vector que une el lugar ocupado por el cuerpo con el origen del sistema de referencia

Este vector, llamado \(r\), se expresa en términos de las coordenadas de un punto \(P\) en el espacio, usualmente expresado como:

\(\vec{r}=\vec{xi}+\vec{yj}+\vec{zi}\)

Donde \(x, y, z\) son los respectivos ejes, y los valores de \(i, j, k\) son los valores de las cooredenadas respectivamente

Esta representación puede adaptarse al plano cartesiano, asumiendo que \(k=0\) por lo que el vector se representaría como \(\vec{r}=\langle xi, yj\rangle\)

Calcular el vector posicion

Encuentra el vector posición y su modulo en el punto \(P(-1,2)\)

Para calcular el vector \(\vec{r}\) en el punto \(P\):

Sustituyendo en \(\vec{r}=xi+yj+zk\) tenemos que:

\(\vec{r}=-1\cdot i+2\cdot j+0\cdot z\)

Teniendo así que \(\vec{r}=\vec{-i}+\vec{2j}\)

Para \(\vec{|r|}\) se calcula:

Sustituyendo \(\vec{|r|}=\sqrt{xi^2+yj^2+zk^2}\) tenemos que:

\(\vec{|r|}=\sqrt{(-1)^2+2^2+0^2}\)

Teniendo así \(\vec{|r|}=\sqrt{5}\)
